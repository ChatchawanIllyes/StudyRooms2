## Get the App Running on Your iPhone in 3 Steps

### Step 1: Install Dependencies
Open your terminal in this project folder and run:
```bash
npm install
```

### Step 2: Start the App
```bash
npx expo start
```

### Step 3: Open on Your iPhone
1. Download "Expo Go" from the App Store on your iPhone
2. Open Expo Go
3. Scan the QR code shown in your terminal
4. The app will load on your phone!

## Troubleshooting

**QR Code not scanning?**
- Make sure your phone and computer are on the same WiFi network
- Try typing the URL manually in Expo Go

**App won't load?**
- Make sure all dependencies installed correctly
- Try running `npm install` again
- Restart the development server with `npx expo start`

**Need help?**
- Check the full README.md for more details
- Visit [Expo Documentation](https://docs.expo.dev/)
